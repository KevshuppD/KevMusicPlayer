# Kev Music Player

**Kev Music Player** is a highly optimized, production-grade music player for Android, built with **Kotlin** and **Jetpack Compose**. 

It focuses on providing a premium user experience with a vibrant **Material Design 3** interface, buttery-smooth **120Hz** animations, and an intelligent library management system.

## 🚀 Features

- **Instant Startup**: Local SQLite caching (Room) ensures your library is ready the moment you open the app.
- **Smart Media Scanning**: Automatically organizes music into folders, albums, artists, and genres.
- **High-Performance UI**: Programmatically enforced 120Hz refresh rate for fluid scrolling.
- **Advanced Navigation**: Fast-scroll alphabet sidebar for large libraries, collapsing into a clean view when not in use.
- **Dynamic Playback**: Powered by AndroidX Media3 with robust background service and persistent playback queue.
- **Integrated Lyrics**: Synchronized LRC support with fetching via LRCLIB, manual editing, and interactive seeking.
- **Intelligent Search**: Compact and efficient search bar that filters your library in real-time.
- **File Management**: Secure physical file deletion with native Android 11+ storage permission handling.
- **Adaptive Layout**: Fully responsive design for phones, tablets, and foldable devices.

## 🛠 Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose (Declarative UI)
- **Design:** Material Design 3
- **Media Engine:** AndroidX Media3 (ExoPlayer)
- **Database:** Room SQLite
- **Architecture:** MVVM (Model-View-ViewModel) + Coroutines & Flow
- **Image Loading:** Coil

## 📦 Getting Started

1. Clone this repository.
2. Open in Android Studio.
3. Build and run the `app` module on your Android device.
